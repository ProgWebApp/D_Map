## Requirements

- **Global** :
  - **Environment** : Docker (dev and prod)
  - **Language** : TypeScript (with Vite)
- **Frontend** :
  - **Framework** : React
  - **CSS Framework** : MUI
  - **Request Engine** : Axios
- **Backend** :
  - **Framework** : Express Koa
  - **ORM** : Mongoose
  - **Database** : MongoDB
